* # GetErrMsg
* Get error description for this errcode
* ## C / CSH
* `int geterrmsg(short int errcode, char *buf);`
* ## ASM
* `>PUSHB errcode`
* `>PUSHW buf`
* `>LIBC GetErrMsg`
* ## RETURN VALUE
*\--------------------------------------
K.GetErrMsg		ldy #0					buf
				jsr RAMSW.GetStkYW
				>STYA A2L
				
				ldy #2					errcode
				jsr RAMSW.GetStkY
				sta A3L
				
				ldx #X_ERR
				jmp RAMSW.JMPX
