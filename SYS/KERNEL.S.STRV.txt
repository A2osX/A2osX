* # StrVGet
* ## ASM
* `>PUSHB hSTRV`
* `>PUSHB index`
* `>PUSHW ptr`
* `>SYSCALL StrVGet`
* ## RETURN VALUE
*  CC: Y,A = Ptr
*  CS: Y,A = NULL
*\--------------------------------------
K.StrVGet		>PULLW FORPNT			ptr

				>PULLA 					index
				eor #$ff
				tax
				
				>PULLA

				sta SETREADAUX
				sta SETWRITEAUX
				jsr STRVX.StrVGet
				sta CLRREADAUX
				sta CLRWRITEAUX
				rts	
*/--------------------------------------

* # StrVAdd
* ## ASM
* `>PUSHB hSTRV`
* `>PUSHW str`
* `>SYSCALL StrVAdd`
